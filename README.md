# Competitive Dynamic Pricing using Multi-Agent Reinforcement Learning

This project studies pricing behaviour in repeated oligopoly markets using Multi-Agent Reinforcement Learning (MARL).

## Files

* `env.py` : Pricing environment and market dynamics
* `agent.py` : Tabular Q-Learning implementation
* `baselines.py` : Baseline agents (Random, Always-50, Always-70, Tit-for-Tat)
* `train.py` : Training and experimentation script

## Work Completed

* Implemented a repeated pricing environment
* Implemented tabular Q-Learning from scratch
* Added epsilon-greedy exploration and epsilon decay
* Conducted experiments with different demand structures
* Compared RL agents against rule-based baselines
* Studied collusion, competition, and equilibrium formation
